How to Use These Icons

  1. Take a screenshot of each icon above (using your system's screenshot tool)
  2. Crop the screenshots to match the exact dimensions of each icon
  3. Save each cropped image with the appropriate filename as listed under each icon
  4. Place these files in your project's public directory to replace the existing icon files
  5. Alternatively, you can also right-click on each icon and use "Save Image As" if your browser supports it

These styled icons match the updated header design of your Al-Quran app and provide a consistent visual identity across all platforms.
